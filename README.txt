로그라이크 엔진으로의 구조


로그라이크 엔진 - 플레이어, 적, NPC, 아이템, 클래스, 던전 클래스, 게임 흐름 제어
                                      |

기반 엔진 - 충돌, 이미지 출력, 게임 진행 사이클, 입력, 그외 기본 데이터


던전 생성기 - 지형 생성 -> 아이템 배치 -> 몬스터 배치

 - 규칙 1. 아이템이 있는 곳에는 몬스터가 같이 있다
 - 규칙 2. 던전은 층 단위로 생성되며, 매개변수로 크기를 입력받는다.
 - 규칙 3. 던전은 방이라는 단위로 나뉘며 크기에 따라 방의 개수가 결정된다.
 - 규칙 4. 던전에는 입구와 출구가 있다.
